What is Eclipse Subversion Export By Revision?
----------------

The plugin will allow you to export files from a project shared by Subversion
based on their revision numbers. Open the export dialog and choose
Team > Subversion Export by Revision.
An export dialog will appear and allow you to define the minimum and maximum
revision numbers of the files to be exported. The files are copied to the
defined output directory.


The plugin can be obtained at the following update site:
http://update.bitexpert.de/eclipse/3.5/
